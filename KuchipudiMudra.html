In Kuchipudi dance, hand gestures are called Hasta Mudras. They help dancers express emotions, actions, and stories through hand movements. There are two main types of mudras used in Kuchipudi dance.

Types of Kuchipudi Mudras

Type of Mudra Description Examples
Asamyuta Hasta Mudras These mudras are performed using a single hand. Pataka, Tripataka, Ardhapataka, Katakamukha, Mayura, Ardhachara,Arala...............
Samyukta Hasta Mudras These mudras are performed using both hands together. Anjali, Kapotam, Karkata, Swasthika, Dola, Pushapaputo, Utsunga,...........
